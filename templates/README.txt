## Usage

### Installation

Add to your +Gemfile+:

```
gem 'my-gem'
```

Then install:

```
bundle install
```

If not using bundler, just use RubyGems: `gem install my-gem`

### Configuration

In `config/initializers/my-gem.rb`:

```ruby
MyGem.configure do |config|
  config.my_config_param = ENV['MY_GEM_CONFIG_PARAM']
end
```
